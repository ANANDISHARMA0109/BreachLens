# BreachLens
**BreachLens** is a cyber forensic analytics tool designed to investigate and reconstruct web server intrusions. It leverages AI-based anomaly detection (like Isolation Forest) and rule-based analysis to identify suspicious behaviors such as directory traversal, brute-force logins, SQL injection, and large file downloads. BreachLens parses server logs, detects stealthy attacks, attributes IP origins via WHOIS and geolocation, verifies file integrity, and visualizes attacker timelines—ultimately generating a comprehensive forensic report for incident response and policy improvement.
